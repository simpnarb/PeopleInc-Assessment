# People Inc - VeryWellHealth assessment

This test using the playwright framework accomplishes the following:
- Going to https://www.verywellhealth.com/
- Clicks on the first component/article
- Verifies the article title matches the title from the component we clicked on

This is tested on:
- Chromium
- Firefox
- Webkit (Safari)

And is reported using:
- Allure
- Playwright report
